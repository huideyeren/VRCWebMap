VRC Web Map

利用規約

制定日: 2026年6月25日

1. 適用

この利用規約は、るいざ・しゃーろっと（以下「運営者」といいます）が提供する VRC Web Map（以下「本サービス」といいます）の利用条件を定めるものです。本サービスを利用する方（以下「利用者」といいます）は、本規約に同意したものとみなします。

2. 本サービスの目的

本サービスは、現実の場所、VRChat ワールド、関連 Web サイト、コメントなどを地図上で参照・共有するためのサービスです。登録された情報は、他の利用者が閲覧できる公開情報として扱われます。

3. アカウントとログイン

本サービスの投稿機能を利用するには Discord ログインが必要です。利用者は、運営者が指定する Discord サーバーに参加している必要があります。管理者権限は、Discord サーバー上の管理者ロールをもとに判定されます。

4. 投稿内容

利用者は、自身が投稿する Spot、VRChat World、場所情報、Web リンク、コメントその他の情報について責任を負います。利用者は、投稿に必要な権利や許可を自身で確認するものとします。

投稿内容は、本サービスの表示、編集、検索、バックアップ、PortalLibrarySystem 向け JSON 出力など、本サービスの提供に必要な範囲で利用されます。

運営者は、投稿内容が本規約に違反すると判断した場合、または運用上必要と判断した場合、事前の通知なく投稿内容を編集または削除できるものとします。削除依頼は問い合わせ先までご連絡ください。
