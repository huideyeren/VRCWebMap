VRC Web Map

プライバシーポリシー

制定日: 2026年6月25日

1. 基本方針

るいざ・しゃーろっと（以下「運営者」といいます）は、VRC Web Map（以下「本サービス」といいます）の提供にあたり、利用者の情報を必要な範囲で取得し、適切に取り扱います。

2. 取得する情報

  • Discord ログインにより取得する Discord ユーザー ID、ユーザー名、表示名、アバター情報
  • 対象 Discord サーバーへの参加状態、本サービス内の管理者権限
  • 利用者が手動登録するVRChat表示名
  • 利用者が投稿した Spot、VRChat World、場所情報、Web リンク、コメント
  • 投稿者を識別するためのユーザー ID
  • ログイン状態を保持するための cookie
  • サーバーログ、アクセス日時、リクエスト情報、エラー情報

ブラウザの位置情報を利用する場合がありますが、現在地そのものはサーバーへ保存しません。ただし、利用者が Spot として登録した緯度・経度は投稿データとして保存されます。

3. 利用目的

  • 本サービスへのログイン、サーバー参加確認、表示名表示、管理者判定のため
  • 投稿内容の表示、編集、削除、投稿者確認のため
  • PortalLibrarySystem 向け JSON など、本サービスの機能を提供するため
  • 不具合調査、荒らし対策、セキュリティ確保、問い合わせ対応のため
  • サービス改善や運用判断のため

4. 外部サービスとの連携

本サービスは、Discord OAuth による認証、OpenStreetMap の地図タイル、VRChat ワールドページ、Web リンクの OGP preview 取得など、外部サービスを利用または参照する場合があります。外部サービスの利用にあたっては、それぞれのプライバシーポリシーや利用規約も適用されます。

Web リンクや VRChat World の preview を取得する際、対象 URL へサーバーからアクセスすることがあります。preview 情報は表示のために一時的に利用し、保存データには含めません。

5. 第三者提供

運営者は、法令に基づく場合、利用者の同意がある場合、または本サービスの安全な運用に必要な場合を除き、取得した個人情報を第三者へ提供しません。ただし、投稿された Spot や関連情報は公開情報として他の利用者に表示されます。

6. 保存期間と削除

取得した情報は、本サービスの運用に必要な期間保存します。投稿内容やアカウントに紐づく情報の削除を希望する場合は、問い合わせ先までご連絡ください。運用上または技術上の理由により、バックアップ等からの完全な削除には時間がかかる場合があります。

7. 安全管理

運営者は、取得した情報の漏えい、滅失、毀損、不正アクセスを防ぐため、必要かつ合理的な安全管理措置を講じます。

8. ポリシーの変更

運営者は、必要に応じて本ポリシーを変更できます。変更後のポリシーは、本サービス上で公開された時点から効力を生じます。

9. 問い合わせ

本ポリシーに関する問い合わせは、maps@inokashiraskey.jp までご連絡ください。
