VRC Web Map

使い方

VRC Web Map は、現実の場所と VRChat ワールドを地図上で結びつけて共有するための地図アプリです。

1. 地図を見る

地図上の marker をクリックすると、登録済み Spot の説明、関連する VRChat World、場所情報、Web リンク、コメントを確認できます。

URL に ?spotId={id}、?spot={id}、または #spot={id} が含まれている場合は、その Spot を中心に表示します。

右上の + と − で細かく拡大・縮小できます。レイヤー切替から標準地図（OpenStreetMap）と航空写真（国土地理院の全国最新写真）を選べます。

2. Spot を検索する

画面上部の「Spot 検索」欄から、スポット名と説明を検索できます。キーワードを入力して「検索」を押すと、条件に一致する Spot だけを地図と一覧に表示します。

空白で区切って複数の語を入力した場合は、すべての語を含む Spot を表示します。検索を解除する場合は「クリア」を押してください。

3. ログインする

Spot や関連情報を登録するには Discord ログインが必要です。ログインできるユーザーは、設定された Discord サーバーに参加しているユーザーに限定されます。

初回ログイン後、メニューの「プロフィール設定」からVRChat内で表示されるDisplay Nameを登録してください。未登録でも地図は閲覧できますが、投稿・編集はできません。

4. Spot を登録する

ログイン後、地図上で登録したい場所を右クリックすると Spot 登録フォームが開きます。Spot 名、説明、緯度・経度、都道府県または地域を入力して保存してください。

説明欄では、見出し、箇条書き、リンクなどの簡単な Markdown を利用できます。

青いピンは通常Spot、オレンジは施設情報あり、紫はVRChatワールドありを示します。両方ある場合は紫を優先します。「現在地に戻る」でブラウザーの現在地へ地図を移動できます。現在地はサーバーへ保存しません。

5. 関連情報を追加する

Spot の詳細画面から、VRChat World、場所情報、Web リンク、コメントを追加できます。VRChat World や Web リンクには、外部ページの preview が表示されることがあります。

6. 編集・削除する

登録者本人または管理者は、登録したデータを更新・削除できます。管理者としてログインしている場合は、管理用画面から Spot と関連情報をまとめて編集できます。

Spot に関連情報が紐づいている場合、Spot の削除は拒否されます。先に関連情報を削除してください。

7. KML/KMZ を取り込む・出力する

地図画面の「KMLで移行・出力」から、Google My Maps や uMap と共通で使えるKMLを扱えます。KML/KMZの取り込みにはログインとVRChat表示名の登録が必要です。

取り込みでは、まず「Preview」で候補を確認します。既存Spotから50m以内の候補は重複の可能性があるため最初は選択されません。内容を確認して必要な候補だけチェックし、「Import」を押してください。実行直前に近くのSpotが増えていた場合は、保存せず再確認を求めます。

Spotを選んでKML出力すると、Google My Maps や uMap へ取り込めます。各Spotには出典、登録者名、元Spotへのリンクを含めます。OpenStreetMap本体へ自動投稿は行いません。1ファイルあたり最大 5,000 件の Point Placemark を読み込めます。KML の coordinates は WGS84 の longitude,latitude[,altitude] として読み込みます。LineString、Polygon、MultiGeometry、NetworkLink、外部参照は現在未対応です。

8. PortalLibrarySystem 用 JSON を出力する

上部メニューの「Portal JSON」から専用画面を開き、PortalLibrarySystem 向けの WorldData.json を生成できます。未ログイン時は地域カテゴリと全体公開カテゴリ、ログイン中はさらに自分の Personal カテゴリを出力します。

既存の WorldData.json を選ぶと、未知の拡張値を保ったまま本システムのカテゴリを末尾へ追加できます。ファイルは 5 MiB 以下にしてください。内容は処理後に保存されません。カテゴリ名の重複や、同名 Role の定義が競合する場合はマージを中止します。

9. 地図外ワールドを管理する

ログイン済みの一般ユーザーは、VRChat表示名を登録すると本人用の Personal カテゴリを複数作成できます。Personal の Role には本人の最新のVRChat表示名を使います。管理者は全体公開の Public カテゴリや、別ユーザーを所有者にした Personal カテゴリも作成できます。

カテゴリの公開範囲と所有者は作成後に変更できません。Personal は所有者または管理者、Public は管理者だけが名前変更、削除、配下ワールドの登録・編集・移動・削除を行えます。
